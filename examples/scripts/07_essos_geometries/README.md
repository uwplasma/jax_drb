# ESSOS-driven geometries

Examples that use ESSOS to generate field-line geometry coefficients and then run the linear solver.

- `stellarator_nearaxis_essos.py`: near-axis stellarator workflow (ESSOS near-axis field).
- `essos_vmec_edge_s09.py`: VMEC field-line example at `s=0.9`.
- `essos_nearaxis_r01.py`: near-axis example at `r=0.1`.
