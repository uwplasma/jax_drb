# Literature-aligned tokamak SOL workflows

These scripts follow (and extend) the workflows in the SOL linear-mode literature:

- Mosetto et al. (2012): regime maps and drift-wave / ballooning branches.
- Halpern et al. (2013): gradient removal and SOL width proxy workflows.
- Jorge et al. (2016): ISTTOK-inspired linear workflow.

Scripts:

- `mosetto2012_driftwave_branches.py`
- `mosetto2012_ballooning_branches.py`
- `mosetto2012_regime_map.py`
- `halpern2013_gradient_removal_lp.py`
- `halpern2013_salpha_ideal_ballooning_map.py`
- `jorge2016_isttok_linear_workflow.py`
