# Linear basics

Quick linear stability workflows using the `jaxdrb-scan` / `jaxdrb-scan2d` machinery.

Scripts:

- `slab_ky_scan.py`: slab ky scan (curvature off, drift-wave-like).
- `circular_tokamak_ky_scan.py`: circular tokamak ky scan (curvature on).
- `salpha_cyclone_ky_scan.py`: Cyclone-like s–α ky scan.
- `cyclone_kxky_scan.py`: 2D scan over `(kx, ky)` in s–α geometry.
- `em_beta_scan.py`: electromagnetic (`psi ~ A_||`) beta scan.
- `hot_ions_tau_scan.py`: hot-ion (`Ti`) scan over `tau_i`.
